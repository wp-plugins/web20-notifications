== Description ==
<ru>Расширенные способы уведомлений о событиях в блоге, поддержка ICQ, Jabber уведомлений о комментариях, входах пользователей, добавлению черновиков и публикации записей.
<en>Advanced event notifications in the blog, support ICQ, Jabber comment notifications, user logins, adding draft and publish records.

== Installation ==

While doing the installation procedure, it is recommended to go through all the steps first before viewing the output. If you don't, you'll get nasty error messages.

= For those with Sidebar Widget compatible themes =

1. Upload folder `web20-notifications` to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
